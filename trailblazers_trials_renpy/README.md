# Trailblazers Trials Ren'Py Prototype

This is a starter Ren'Py project for the Trailblazers universe. It is set up as a small narrative RPG / visual novel hybrid so the story can move first, while RPG systems stay simple and optional.

## What Is Included

- A Ren'Py-compatible `game/` folder.
- Source context copied into `game/data/source_context/`.
- Curated sprite copies with clean filenames under `game/images/`.
- A first playable chapter in `game/chapters/chapter_01.rpy`.
- A placeholder light battle scene in `game/systems/simple_battle.rpy`.
- Starter variables for trust, reputation, dragon pressure, and mana strain.

## How To Open

1. Download and open Ren'Py.
2. In the Ren'Py launcher, choose the project directory:
   `trailblazers_trials_renpy`
3. Launch the project.

If Ren'Py does not detect it automatically, create a new Ren'Py project through the launcher, then copy this project's `game/` folder over the generated `game/` folder.

## Prototype Goal

The current prototype focuses on one question:

Can Oren's dual-soul premise carry a story-first RPG opening with only light systems?

The next pass should improve asset presentation, expand the opening scene, and decide how much RPG mechanics should exist around the story.
